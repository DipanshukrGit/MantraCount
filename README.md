# MantraCount 🕉️

![MantraCount Banner](./assets/images/banner.png)

## 📱 About MantraCount

**MantraCount** is a peaceful and easy-to-use mobile app built with **Expo** where users can count mantra chants, search for different mantras, and listen to soft meditation music.

The app is designed to help users stay focused during jaap, meditation, and daily spiritual practice.

---

## ✨ Features

- 🧘 Count mantra chants easily
- 🔍 Search and explore different mantras
- 🎵 Play soft meditation music
- 📱 Clean and simple mobile UI
- 🌙 Peaceful user experience
- ⚡ Built with Expo for smooth performance
- 🎯 Helpful for jaap, meditation, and focus

---

## 🛠️ Tech Stack

- React Native
- Expo
- JavaScript
- NativeWind / CSS Styling
- Expo Audio
- React Navigation / Expo Router
